## Слайд A · «Как выбирается слово» · тип C

**Заголовок:** Как выбирается слово

**Визуал (≈62%):** незаконченная фраза, под ней — ранжированные кандидаты с полосками вероятности. Верхний выделен как выбранный. Внизу — лента «слово за словом», где из выборов складывается строка.

```
В ельнике почва обычно  ▮___
──────────────────────────────────────
кислая    ████████████   58%   ← выбрано
влажная   █████           22%
бедная    ███             11%
тёплая    █                6%
…прочие   ▏                3%
──────────────────────────────────────
…и так, слово за словом →
«В ельнике почва обычно кислая, потому что…»
```

**Текст-блок (≈38%):** «Модель оценивает, насколько вероятно каждое следующее слово, и берёт из верхних. И так — слово за словом, до конца ответа.»

**Один акцент:** ответ собирается по одному слову; каждое — выбор из вероятных.
**Подача:** не «знает ответ», а «оценивает вероятность». Эхо T9 (слайд 08): там кандидаты брались из словаря телефона, здесь — из всего языка.

## Слайд B · «Тот же вопрос — разный ответ» · тип D

**Заголовок:** Тот же вопрос — разный ответ

**Визуал:** один запрос сверху → две стрелки вниз → два прогона рядом. Смысл одинаковый, формулировки разные. Различающиеся слова — лёгкая заливка (не волнистое подчёркивание — оно зарезервировано под ошибки).

```
        ┌────────────────────────────────────┐
        │  Чем засуха опасна для ельника?     │   один и тот же вопрос
        └────────────────────────────────────┘
              │                          │
          Прогон 1                    Прогон 2
   ┌─────────────────────┐    ┌─────────────────────┐
   │ Засуха ослабляет    │    │ В засушливый период │
   │ ели: падает прирост,│    │ ели теряют влагу —  │
   │ растёт риск усыхания│    │ прирост снижается,  │
   │ и вспышек короеда.  │    │ выше уязвимость     │
   │                     │    │ к короеду.          │
   └─────────────────────┘    └─────────────────────┘
            оба верны · формулировки разные
```

**Текст-блок / подпись:** «Выбор из вероятных — значит ответ каждый раз немного другой. Оба верны, но не идентичны. Сослаться на „модель сказала" как на фиксированный факт нельзя — это вопрос воспроизводимости.»

**Один акцент:** недетерминизм → воспроизводимость.
**Подача:** «ответ варьируется», не «модель передумала». Температуру/настройку случайности здесь **не называем** — это рычаг Секции 2.

## Слайд 1 · «Как училась модель» · тип C

**Заголовок:** Как училась модель

**Визуал (≈62%) — конвейер слева направо:**

```
   ИСТОЧНИКИ              ОБУЧЕНИЕ                КАРТА СМЫСЛОВ
                                                  
  сайты    ─┐                                      • ель • сосна
  книги    ─┤        ┌──────────────┐              • пихта
  статьи   ─┼──────► │  модель сама │ ──────►      
  код      ─┤        │ подстраивается│             • почва • грунт
  форумы   ─┘        └──────────────┘              • кислотность
                                                  
                  один раз · месяцы ·          (та самая карта
                  тысячи видеокарт              со слайда 25)
```

Слева — поток разных текстов. В центре — коробка «модель подстраивается». Справа — мини-версия карты смыслов со слайда 25, та же кластеризация (ель/сосна/пихта рядом, почва/кислотность отдельно).

**Текст-блок (≈38%):** «Модель не писали по правилам. Ей показали огромное количество текста — и она сама подстроилась под него. Так и сложилась карта смыслов: близкие по смыслу слова встали рядом. Это делается один раз и очень долго — месяцы работы тысяч видеокарт.»

**Один акцент:** карту никто не расставлял руками — она сложилась сама из текста, за один долгий заход.
**Подача:** «подстроилась / сложилась», не «выучила / поняла». «Цена» живёт подписью под коробкой, отдельного слайда под неё нет.

## Слайд 2 · «После обучения карта застывает» · тип C

**Заголовок:** После обучения карта застывает

**Визуал (≈62%) — карта + заморозка + отскок вопросов:**

```
            ❄  карта смыслов застыла  ❄
        ┌───────────────────────────────┐
        │   • ель • сосна • пихта       │
        │   • почва  • кислотность      │  🔒
        │   • засуха • короед           │
        └───────────────────────────────┘
                      ▲
            ваши вопросы  ──►  ✕  (карта не меняется)
```

Та же карта, но «подмёрзшая» (иней/замок поверх — точки на местах, не двигаются). Снизу — стрелка «ваши вопросы» упирается в карту и не меняет её.

**Текст-блок (≈38%):** «Когда обучение закончилось — карта застывает. Дальше модель только пользуется тем, что уже сложилось, и сама по себе не меняется. Сколько бы вы её ни спрашивали — карта остаётся прежней.»

**Один акцент:** модель не меняется от того, что вы ей пишете.
**Мост:** → Часть III, «куда тогда уходят ваши данные» (если не в модель — то куда).
**Подача:** «застывает / не меняется», не «запоминает / забывает».
