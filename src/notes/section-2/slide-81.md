# Слайд 81 · «Галлюцинация вживую»

> Часть V · 5.5 Галлюцинации и разброс · Тип: C
> Хронометраж: ~1 мин · Ведущий: Иван

## Демо-промт (вставляется вживую)

```
Каким ГОСТом регулируется защита ельников от
короеда-типографа? Укажи номер и год.
```

## Что показать

Модель уверенно выдаёт правдоподобный, но выдуманный номер (на слайде подчёркнут волнистой линией). Если ответит корректно/откажется — тоже материал: «иногда срабатывает осторожность; полагаться нельзя».

## Опорная фраза

«Где обучающих текстов много — факт. Где мало — модель достраивает правдоподобное тем же уверенным тоном. Уверенность тона не равна правильности.»
