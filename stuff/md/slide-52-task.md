# Слайд 52 · Где проверять обязательно

## Метаданные
- Тип: D (сравнение в двух колонках)
- Подблок: 3.4 Точность и галлюцинации (финальный слайд подблока)
- Хронометраж: 2 минуты
- Ведущий: Иван

## Контент слайда

### Заголовок
Где проверять обязательно

### Подзаголовок (под заголовком, серый `--mute`)
Развёрнутое правило для работы

### Левая колонка

**Заголовок колонки:** Обязательно проверять

**Пункты** (каждый пункт: основной текст `--ink`, пример в скобках курсивом и `--mute`):

- Цифры и статистика *(15%, −20 °C, 1 200 га)*
- Цитаты и ссылки *(Иванов, 2019, DOI)*
- Нормативные документы *(ГОСТы, статьи закона, СНиПы)*
- Имена, даты, факты *(год выхода работы, авторство метода)*

### Правая колонка

**Заголовок колонки:** Можно доверять

**Пункты** (тот же формат — основной текст `--ink`, пример курсивом `--mute`):

- Объяснения известных понятий *(что такое регрессия, фотосинтез)*
- Переформулировки *(сократить, упростить, перевести)*
- Структура черновика *(оглавление, разделы, тезисы)*
- Поиск аналогий и идей *(подобрать сравнение, варианты)*

### Мета-тег подблока сверху
Нет (по аналогии с 49–51, для единства финальной трети Секции 1)

## Визуал

Без иллюстрации. Чистая типографическая таблица из двух колонок.

- Заголовок и подзаголовок сверху
- Две колонки с тонкой 1px-линией `--rule` между ними
- В каждой колонке: заголовок колонки → 4 пункта построчно
- Каждый пункт: основной текст (`--ink`, обычный вес) + пример в скобках (курсив, `--mute`)
- Между пунктами достаточный вертикальный воздух — аудитория должна успевать прочитывать каждую строку

Размер пунктов средний, ориентир 28–34px. Заголовки колонок чуть крупнее или другого веса, чтобы отличаться от пунктов.

Колонки симметричны: равная ширина, равная высота, одинаковое количество пунктов (4+4). Пункты выровнены по верху колонки, не разъезжаются.

## Композиция слайда

D-шаблон:
- Заголовок сверху
- Подзаголовок мельче, серым
- Две колонки одинаковой ширины с разделительной линией `--rule`
- В каждой колонке: заголовок → 4 пункта
- Без summary внизу
- Без `--accent`, без иконок (галочки, крестики, ⚠️ — всё запрещено)
- Без цветовой маркировки зон (красная/зелёная) — структура слайда работает на типографике

## Чего избегать
- Не выделять левую колонку красным/опасным, правую — зелёным/безопасным. Обе колонки одинаково спокойные.
- Не добавлять иконки (галочки, крестики, восклицательные знаки, лупа) — слайд работает только на типографике
- Не использовать `--accent`
- Не добавлять bullet-маркеры (•, –, →) в начале строк — пункты идут как строки таблицы, не как буллет-лист
- Не делать примеры в скобках слишком длинными — это короткие иллюстрации, не определения
- Не нарушать баланс 4 + 4 — равное количество пунктов критично для смысла слайда
