## Визуал (слева, ~62%)

Стилизованная SVG-«лестница» из 6 ступеней, восходящая снизу вверх слева направо. Дополнительно — вертикальный индикатор «масштаб предсказания» слева от лестницы.

### Шесть ступеней лестницы (снизу вверх)

| # | Подпись на ступени | Положение |
|---|---|---|
| 1 | **Т9** | низ-лево |
| 2 | **Автодополнение** | |
| 3 | **Transformer** | |
| 4 | **GPT-3** | |
| 5 | **ChatGPT** | |
| 6 | **R1** | верх-право |

> Только название. Без годов на слайде. Без логотипов. Без иллюстраций (Nokia, лого OpenAI, скриншоты и т.п.) — только текст.

### Индикатор «масштаб предсказания» (слева от лестницы)

Вертикальная стрелка (или просто вертикальная ось с делениями), указывающая снизу вверх. На делениях — короткие подписи, выровненные по высоте соответствующих ступеней:

| Высота ступени | Подпись на оси |
|---|---|
| Т9 | буква |
| Автодополнение | слово |
| Transformer | фраза |
| GPT-3 | текст |
| ChatGPT | диалог |
| R1 | рассуждение |

> Подписи на оси — IBM Plex Mono, `--mute`, мельче подписей на ступенях. Это служебная шкала, не главное содержание. Стрелка/ось — `--rule` или `--mute`, тонкая.

### Главный визуальный приём

Зритель за секунду охватывает взглядом всю пройденную дистанцию. Слева — что предсказывает модель (от буквы к рассуждению), снизу-вверх — этапы технологии (от Т9 к R1). Две оси читаются параллельно: ось абстракции (что предсказываем) растёт синхронно с осью технологии (чем предсказываем).

### Технические требования к визуалу

- Никаких иллюстраций ВНУТРИ ступеней. Только текстовая подпись
- Никаких логотипов, скриншотов, иконок
- Никаких 3D-эффектов, теней, градиентов — плоская схема
- Ступени — тонкие линии или плоские блоки. Цвет: контур `--rule` или `--mute`, можно лёгкая заливка `--hi`
- Текст подписей на ступенях: IBM Plex Sans, `--ink`, средний размер
- Текст подписей на оси: IBM Plex Mono, `--mute`, мелкий размер
- Стрелка-ось слева: тонкая (1–2px), `--rule` или `--mute`. Если со стрелочным наконечником — небольшой, не доминирующий
- Высота между ступенями — равная. Не моделируем «реальные» временные интервалы (т.к. Т9 и автодополнение — десятилетия, остальные — годы). Это абстрактная лестница масштаба, не таймлайн

### Композиция визуальной области

Лестница занимает основную часть. Ось-индикатор «буква → рассуждение» — узкая колонка слева. Между ними небольшой воздух.

Финальная (верхняя) ступень не должна упираться в верхний край области — пусть есть воздух сверху, чтобы лестница «не упирается в потолок».
