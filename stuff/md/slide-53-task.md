# Слайд 53 · AI берёт на себя задачи, не роль

## Метаданные
- Тип: B (тезис, расширенный центральный блок)
- Подблок: 3.5 Границы применения (первый слайд подблока)
- Хронометраж: 1.5 минуты
- Ведущий: Иван

## Контент слайда

### Заголовок
AI берёт на себя задачи, не роль

### Центральный блок (три строки одинакового веса и цвета)

Строка 1:
> 30 лет назад программы для статистики вызывали те же вопросы.

Строка 2:
> Они перераспределили задачи.

(интервал — пустая строка / увеличенный отступ)

Строка 3:
> Приход Excel не упразднил бухгалтерию — изменил работу.

### Мета-тег подблока сверху
Нет (по аналогии с 49–51, для единства финальной трети Секции 1)

## Визуал

Без иллюстрации. Чистая типографическая композиция:
- Заголовок сверху на стандартном месте B-типа
- Центральный блок из трёх строк, оптически по центру слайда
- Все три строки одного веса (400/500) и цвета `--ink`

Размер строк центрального блока крупнее body, но мельче типичного B-тезиса. Ориентир — 32–40px (вместо стандартных 44–56px), чтобы три строки уместились без визуального давления и было ощущение спокойной констатации, а не «крика».

Тире в строке 3 — типографское `—` с пробелами вокруг.

Между строкой 2 и строкой 3 — увеличенный отступ (примерно 1.5–2× от межстрочного), чтобы Excel-фраза читалась как отдельный смысловой абзац, не как продолжение мысли про статпакеты.

## Композиция слайда

B-тип с расширенным центральным блоком:
- Заголовок сверху
- Три строки текста в центре, равные по весу и цвету
- Без анкора, без иллюстраций, без иконок
- Без `--accent`

## Что переиспользовать
- B-каркас (заголовок сверху + центральный блок)
- Шрифтовая шкала из готовых B-слайдов
- Цветовые токены: `--ink` для всех трёх строк

## Чего избегать
- Не делить центральный блок на колонки — это не сравнение, а единый тезис с двумя историческими опорами
- Не выделять «Excel» или «статпакеты» цветом/жирным — все три строки равноценны
- Не использовать `--accent`
- Не добавлять иконки (диаграмма, калькулятор, символ Excel) — слайд работает только на тексте
- Не использовать слово «профессии» в финальной формулировке — заголовок сменён на «роль» намеренно
