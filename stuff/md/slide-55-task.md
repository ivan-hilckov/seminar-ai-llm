## Визуал

Без иллюстрации. Чистая типографическая карта.

Структура:
- Заголовок сверху
- Подзаголовок мельче, серым
- Три блока вертикально, один под другим, занимают центральную область слайда
- Каждый блок:
  - Заголовок блока («Часть I · Откуда взялись современные модели») — крупнее пунктов, основной цвет `--ink`
  - Под заголовком — список пунктов с маркерами-кружками слева

Маркеры — маленькие заполненные кружки цвета `--mute`, 4–6px, выровненные по базовой линии текста пункта. Не точки, не буллеты `•`, не дефисы — именно небольшие визуальные «отметки», не «список».

Между пунктами внутри блока — небольшой отступ.
Между блоками — увеличенный отступ (1.5–2× межпунктового), чтобы блоки читались как отдельные единицы.

Размеры (ориентир):
- Заголовок слайда — стандартный для C
- Подзаголовок — ~22–26px, `--mute`
- Заголовки блоков (Часть I/II/III) — ~26–30px, `--ink`, вес 500/600
- Пункты подблоков — ~20–24px, `--ink`, обычный вес
- Между блоками увеличенный воздух

Часть I — 6 пунктов (самая длинная), Часть II — 3 пункта, Часть III — 5 пунктов. Не выравнивать искусственно — естественное распределение отражает реальный объём.

Без `--accent`, без иконок, без галочек ✓ или крестиков ✗. Слайд работает на типографике и геометрии списка.

## Композиция слайда

Слайд занимает центр канваса:
- Заголовок и подзаголовок прижаты к верхней рабочей линии
- Три блока распределены по вертикали с равными зазорами между собой
- Левое выравнивание всех текстовых элементов (заголовки блоков, пункты, маркеры)

Все три блока — одной ширины колонки. Длинные строки (например, «Главный тезис: это не интеллект, это статистика») не переносятся — слайд достаточно широкий для них.

## Чего избегать
- Не использовать галочки ✓ — даже с серым цветом они читаются как «выполнено», что превращает карту в чеклист. Нам нужна нейтральная карта пройденного, не отчёт о выполнении.
- Не использовать иконки (флажок, лупа, домик) перед заголовками блоков
- Не использовать `--accent` — финальный слайд должен быть спокойным, не «парадным»
- Не выделять «Главный тезис: это не интеллект, это статистика» цветом или жирным — это пункт наравне с остальными. На этом слайде keystone не выделяется — на 56 он будет в фокусе.
- Не добавлять номера Часть I / II / III как римские цифры италиком (это формат A-слайдов разделителей) — здесь обычный текст «Часть I · …»
- Не разносить блоки по колонкам — структура вертикальная, как в обзоре
- Не делать кружки крупными или яркими — они визуальный шум, не контент
