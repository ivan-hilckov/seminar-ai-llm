## 3.1 Данные и приватность

### 37 · Облачно vs локально (D)

**Ключевое:** запрос либо уходит на чужой сервер, либо остаётся на вашем компьютере.

Две колонки «Облако / Локально», 4 строки сравнения: *что происходит, где обрабатывается, кто видит, что нужно*. Снизу итог «Для чувствительных данных — локально». Без иконок и названий сервисов.

### 38 · Как отключить обучение на ваших данных (C)

**Ключевое:** ваши данные можно исключить из обучения — это переключатель в настройках.

Реконструкция панели настроек «Data controls» с тогглом «Improve the model for everyone» в положении **OFF**. Справа — путь `Settings → Data controls → Improve the model for everyone`.

## 3.2 Память и контекст в работе

### 39 · Память (Memory) — отдельная функция (C)

**Ключевое:** не модель вас запоминает — продукт хранит заметки и подмешивает их в промт.

Слева панель «Заметки о пользователе» с 4 чипами (работает в ВНИИЛМ, интересуется засухой, короткие ответы, зовут Иван) → стрелка в блок «Системные инструкции» (там курсивом «+ заметки о пользователе») → стрелка вниз в блок «Модель». Подпись: «Можно выключить · Можно увидеть · Можно стереть».

### 40 · Когда Память полезна, когда мешает (B)

**Ключевое:** память хороша для предпочтений и вредна для фактов.

Крупный парный антитезис по центру: «Полезна для предпочтений. / Мешает для фактов.» Без визуала.

### 41 · Новый чат — новый лист (D)

**Ключевое:** в новом чате модель не помнит ни имени, ни прошлых вопросов.

Два окна чата: слева «Помнит всё» (история из 5 реплик про план эксперимента по засухе), справа «Не знает ничего» (пустое окно, курсор в поле ввода). Снизу: «Исключение — функция Памяти».

## 3.3 Доступ к интернету

### 42 · Поиск — отдельная функция (C)

**Ключевое:** по умолчанию модель отвечает по памяти обучения; в интернет идёт только по кнопке.

Две сцены интерфейса DeepSeek: вверху кнопка Search выключена → стрелка к книге «знания до 2024» / «модель вспоминает»; внизу Search нажат (чёрный) → стрелка к глобусу «интернет, сейчас» / «модель ищет». Меняется только нажатость кнопки.

## 3.4 Точность и галлюцинации

### 43 · Не знает фактов — знает статистику текстов (B)

**Ключевое:** модель не хранит факты, а достраивает их по статистике; пустоты заполняет похожим на правду.

Заголовок + крупный тезис по центру «Модель не хранит факты — она восстанавливает их по статистике текстов» + бледный якорь снизу. Без визуала.

### 44 · Как выглядит галлюцинация (D)

**Ключевое:** уверенный, безупречный на вид ответ — и без источника.

Два «окна ChatGPT»: слева реальный кейс *Mata v. Avianca, 2023* (выдуманные судебные дела, штраф $5000), справа реконструированный вымышленный *ГОСТ Р 12345-2021* про защиту ельников от короеда. Выдуманные реквизиты подчёркнуты тонкой волнистой линией. Итог: «ответ выглядит безупречно — и не имеет источника».

### 45 · У думающих моделей выше галлюцинации (D)

**Ключевое:** чем длиннее рассуждение, тем больше места для ошибок.

Две колонки с крупными числами: DeepSeek R1 (думающая) — **14.3%** против DeepSeek V3 (обычная) — **3.9%** галлюцинаций. Контраст создаётся размером цифры. Источник — Vectara HHEM Leaderboard.

### 46 · Где проверять обязательно (D)

**Ключевое:** практическое правило — что проверять руками, а чему можно доверять.

Таблица 4+4. Слева «Обязательно проверять»: цифры/статистика, цитаты/ссылки, нормативные документы, имена/даты/факты (с примерами). Справа «Можно доверять»: объяснения понятий, переформулировки, структура черновика, аналогии/идеи.

## 3.5 Границы применения

### 47 · AI берёт на себя задачи, не роль (B)

**Ключевое:** ИИ перераспределяет задачи, а не упраздняет профессию.

Заголовок + три строки по центру: «30 лет назад программы для статистики вызывали те же вопросы» / «Они перераспределили задачи» / «Приход Excel не упразднил бухгалтерию — изменил работу». Без визуала.

### 48 · Она понимает буквально — не интуитивно (B)

**Ключевое:** модель делает написанное, а не задуманное → формулировка это навык. *(Финальный слайд, мост к методике.)*

Заголовок + крупный тезис «Модель выполняет то, что написано, а не то, что вы имели в виду» + две рамки: [Размыто: «Напиши что-нибудь про лес»] | [Конкретно: развёрнутый запрос про введение статьи о защите ельников]. Якорь снизу: «формулировка запроса — навык, а не магия».
